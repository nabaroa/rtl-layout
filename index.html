Optimise your layout for RTL languages

  • Alignment
  • Spacing
  • Columns

Usage

Set dir attribute in HTML tag to display proper layout direction. 


<html dir="rtl" lang="en"></html>

The browser will show a proper layout if you follow the best practices for directional layout building. Try the buttons on header to test.

Best practices

Don't

  • Bootstrap column system or similar based in floats
  • Horizonal padding/margin for spacing purposes
  • Text alignment based on right/left

Do

  • Grid/flex containers for layouts
  • Use gap for spaces between elements
  • Use flex aligment instead of text alignment

Here an example of how to modify your directional classes. Take into account that what is left must be converted to right and vice versa in RTL direction so we can create conditional classes using the cascade:


html[dir="rtl"] {
  .text-align-left {
    text-align: right;
  }
}
      
    
.my-component {
  margin-right: 24px;

  html[dir="rtl"] &{
    margin-right: initial;
    margin-left: 24px;
  }
